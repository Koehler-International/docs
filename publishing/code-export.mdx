## Overview

MobileCoder gives you **full ownership of your code**. You can download your complete project at any time and continue development in your preferred environment.

## Downloading Your Project

### Download as ZIP

1. Open your project in MobileCoder
2. Click the **Download** button in the project header
3. Your complete codebase will download as a ZIP file

The download includes everything you need to run and publish your app.

## What's Included

Your exported project contains:

| Content | Description |
|---------|-------------|
| **Source Code** | All React Native/Expo TypeScript files |
| **package.json** | Dependencies and scripts |
| **app.json** | Expo configuration |
| **Assets** | Images, fonts, and other static files |
| **Navigation** | Complete Expo Router structure |
| **Styling** | NativeWind/Tailwind configuration |

## Running Your Exported Code

After downloading and extracting your project:

```bash
# Navigate to your project
cd your-project

# Install dependencies
npm install

# Start the development server
npx expo start
```

You can then:

- Press `i` to open in iOS simulator
- Press `a` to open in Android emulator
- Scan the QR code to open in Expo Go

## Development Options

Once you have your code, you can work with it in any environment:

### VS Code / Cursor

Open the project folder in your favorite code editor for full control over the codebase.

### Xcode

Generate native iOS project files and use Apple's development tools:

```bash
npx expo prebuild -p ios
xed ios
```

### Android Studio

Generate native Android project files:

```bash
npx expo prebuild -p android
```

### Continue with Expo

Keep using Expo's managed workflow for cross-platform development.

## Frequently Asked Questions

<AccordionGroup>
  <Accordion icon="code" title="Do I own the code?">
    Yes, you own all code generated by MobileCoder. There are no restrictions on how you use, modify, or distribute it.
  </Accordion>
  <Accordion icon="arrows-rotate" title="Can I import changes back to MobileCoder?">
    Currently, MobileCoder focuses on building within the platform. Changes made locally won't sync back. However, your GitHub repository always has the latest code from MobileCoder.
  </Accordion>
  <Accordion icon="file-zipper" title="Why is the download a ZIP file?">
    ZIP files ensure you get all files intact, including hidden configuration files and folder structures that might be missed with other download methods.
  </Accordion>
</AccordionGroup>
